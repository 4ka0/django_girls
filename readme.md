# Django Girls Blog

### Overview

A blog built following the [Django Girls tutorial](https://tutorial.djangogirls.org/en/).</br>

[Live demo](https://jjl-djangogirlsblog.herokuapp.com) (deployed on Heroku).

### Features

Uses ...</br>

have a blog where you

need a username and password to log in,
need to be logged in to add, edit, publish or delete posts,
and can log out again.


### Built using:

* Python 3.7
* Django 3.0.8
* django-crispy-forms 1.9.2
* Gunicorn 20.0.4
* python-decouple 3.3
* Bootstrap 4
* Visual Studio Code 1.47.3
* macOS 10.14.6
* Heroku
* Pipenv
